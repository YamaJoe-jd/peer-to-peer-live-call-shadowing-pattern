# Security Policy

## Supported Versions

The following versions of Network File Guard receive security updates. Older versions may continue to function but are not maintained for security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Active          |
| 0.x     | ❌ Unsupported     |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately and responsibly.

**How to report:**
- Email: SOON! (Please report vulnerabilities using GitHub’s private security advisory workflow.)  
- Include: steps to reproduce, affected version(s), expected vs. actual behavior, and any relevant logs or proof‑of‑concept details.

**What to expect:**
- Acknowledgment within **48 hours**  
- Status update within **5 business days**  
- If confirmed, a coordinated fix and responsible disclosure timeline  
- Credit in release notes unless anonymity is requested

Thank you for helping keep Network File Guard secure.
